## 트러블슈팅

### DLL 로드 실패 (exit code 0xc0000135)

`$env:PATH`에 vcpkg DLL 경로가 포함되지 않았을 때 발생한다. Debug 빌드는 `debug/bin/` 경로의 DLL을 사용한다.

### Redis 연결 실패

`docker compose up -d redis`로 Redis가 실행 중인지 확인한다. 기본 포트 6379.

### Chat Server 연결 거부 (Room Server 로그)

Chat Server가 Room Server보다 먼저 기동되어야 내부 채널(:8083) 연결이 즉시 성공한다. Room Server는 5초 간격으로 재연결을 시도하므로 순서가 바뀌어도 동작한다.

### Proto 파일 변경 후

`.proto` 파일을 수정했으면 C++ 빌드(`cmake --build build`)와 Python 코드젠(`protoc --python_out=...`)을 모두 다시 실행해야 한다.
