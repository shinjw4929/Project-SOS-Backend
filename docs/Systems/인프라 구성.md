# 인프라 구성

Docker Compose 기반 인프라 서비스 상세.

---

## 서비스 구성

```
docker-compose.yml
├── redis          Redis 7 Alpine          :6379
├── clickhouse     ClickHouse 24.3         :8123 (HTTP), :9000 (Native)
├── vector         Vector 0.41.1           (포트 없음, 내부 전용)
├── grafana        Grafana 11.4.0          :3000
├── room-server    Dockerfile (room)       :8080, :8081
└── chat-server    Dockerfile (chat)       :8082, :8083
```

### 서비스 의존성

```
redis (healthy) ──┬── chat-server
                  └── room-server (+ chat-server started)

clickhouse (healthy) ──┬── vector
                       └── grafana
```

## ClickHouse

| 항목 | 값 |
|------|------|
| 이미지 | clickhouse/clickhouse-server:24.3 |
| 포트 | 8123 (HTTP), 9000 (Native) |
| 볼륨 | clickhouse_data, init.sql, users.xml |
| 인증 | default / `${CH_PASSWORD:-dev123}` |
| 헬스체크 | clickhouse-client 쿼리 (5초 간격, 10회 재시도) |

### 스키마: `project_sos.service_events`

| 컬럼 | 타입 | 용도 |
|------|------|------|
| timestamp | DateTime64(3, 'UTC') | 이벤트 시각 |
| service | LowCardinality(String) | game / room / chat |
| level | LowCardinality(String) | DEBUG / INFO / WARNING / ERROR |
| category | LowCardinality(String) | Room, Chat, Room:Internal 등 |
| message | String | 로그 메시지 본문 |
| context | String | 추가 컨텍스트 (JSON) |

- 파티션: (service, date)
- TTL: 90일 자동 삭제
- Materialized View: violation_rate, error_rate (시간별 집계)

### 스키마: `project_sos.chat_messages`

| 컬럼 | 타입 | 용도 |
|------|------|------|
| timestamp | DateTime64(3, 'UTC') | 메시지 시각 |
| session_id | String | 게임 세션 ID (LOBBY는 빈값) |
| channel | LowCardinality(String) | LOBBY / ALL / WHISPER |
| sender_id | String | 발신자 ID |
| sender_name | String | 발신자 표시명 |
| target_id | String | 귓속말 대상 ID (WHISPER 전용) |
| content | String | 메시지 내용 |

- 파티션: date
- ORDER BY: (session_id, timestamp)
- TTL: 90일 자동 삭제

## Vector

| 항목 | 값 |
|------|------|
| 이미지 | timberio/vector:0.41.1-debian |
| 볼륨 | vector.toml, docker.sock (ro), game logs (ro), vector_data |
| 버퍼 | disk, 512MB, when_full: block |
| 배치 | max_events: 1000, timeout: 5초 |

### 파이프라인

```
Sources:
  game_logs (file) ──→ parse_game ──→ clean_fields ──→ clickhouse sink (service_events)
  docker_logs ──→ parse_cpp ──→ route_cpp
                                  ├── chat_message (Chat:Message) ──→ extract_chat_fields ──→ clickhouse_chat_messages sink (chat_messages)
                                  └── _unmatched ──→ clean_fields ──→ clickhouse sink (service_events)
```

### 파싱 정규식

- **parse_game**: Game Server 로그 (world, category, network_id, ghost_id 추출)
- **parse_cpp**: C++ 서버 로그 (category 추출, `[\w:]+` 패턴으로 콜론 포함 카테고리 지원)

## Grafana

| 항목 | 값 |
|------|------|
| 이미지 | grafana/grafana:11.4.0 |
| 포트 | 3000 |
| 플러그인 | grafana-clickhouse-datasource |
| 데이터소스 | ClickHouse (Native :9000) |

### 대시보드

| 대시보드 | 패널 | 대상 |
|---------|------|------|
| Service Overview | 9 | 전체 서비스 (로그 볼륨, 레벨/카테고리 분포) |
| Game Events | 7 | Game Server (Wave, 킬, 경제, 스폰) |
| Room Metrics | 8 | Room Server (방 생성/제거, 입퇴장, 토큰, 에러) |
| Chat Metrics | 9 | Chat Server (접속, 인증, 세션, 채널별 메시지, 메시지 로그, 에러) |

### 알림 규칙 (5개)

| 규칙 | 조건 | 심각도 |
|------|------|--------|
| Error Rate Spike | 5분간 ERROR 10건+ | critical |
| Room Creation Failure | 5분간 Room ERROR 5건+ | warning |
| Redis Connection Error | 5분간 Redis 에러 3건+ | critical |
| Chat Server Disconnected | 5분간 Room:Chat 경고 3건+ | warning |
| Internal Channel Error | 5분간 Internal 에러 3건+ | warning |

## Dockerfile

멀티 스테이지 멀티 타겟 빌드.

```
build-base (Ubuntu 24.04)
  ├── g++, cmake, ninja, vcpkg
  ├── vcpkg 의존성 설치
  └── cmake --preset=default && cmake --build build

room-server (Ubuntu 24.04 minimal)
  ├── 바이너리: build/src/room/room-server
  ├── 설정: config/server_config.json
  └── CMD: room-server config/server_config.json

chat-server (Ubuntu 24.04 minimal)
  ├── 바이너리: build/src/chat/chat-server
  ├── 설정: config/server_config.json
  └── CMD: chat-server config/server_config.json
```
