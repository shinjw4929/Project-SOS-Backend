## 심각도 기준

| 심각도 | 기준 | 예시 |
|--------|------|------|
| **Critical** | 런타임 크래시, 데이터 레이스, 보안 취약점, 서비스 중단 | 메모리 누수, 인증 우회, 소켓 미닫힘 |
